You can manage Laminar datasets entirely from the command line using `lmnr-cli` — a language-agnostic CLI that works with any project, regardless of language.

```bash
npx lmnr-cli@latest dataset list
npx lmnr-cli@latest dataset create my-dataset data.jsonl -o my-dataset.jsonl
npx lmnr-cli@latest dataset push data.jsonl -n my-dataset
npx lmnr-cli@latest dataset pull output.jsonl -n my-dataset
```

The CLI supports JSON, JSONL, and CSV formats, and provides a `--json` flag for structured output suitable for piping to other tools or AI agents.
