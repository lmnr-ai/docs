# TypeScript Types & Constants

## LaminarAttributes
LLM span attribute keys: `PROVIDER`, `REQUEST_MODEL`, `RESPONSE_MODEL`, `INPUT_TOKEN_COUNT`, `OUTPUT_TOKEN_COUNT`, `TOTAL_TOKEN_COUNT`, `INPUT_COST`, `OUTPUT_COST`, `TOTAL_COST`, plus helpers like `SPAN_INPUT`, `SPAN_OUTPUT`, `SPAN_TYPE`, `SESSION_ID`, `USER_ID`, `TRACE_TYPE`.

## Types
`ChatMessage`, `NodeInput`, `Event`, `InitEvaluationResponse`, `Dataset`, `PushDatapointsResponse`, `EvaluationDatapointDatasetLink`, `EvaluationDatapoint`, `GetDatapointsResponse`, `SemanticSearchResult`, `SemanticSearchResponse`, enums (`SpanType`, `TraceType`, `TracingLevel`), `LaminarSpanContext`, `MaskInputOptions`, `SessionRecordingOptions`, `Span` (re-export), `DatasetType`.
