# SDK Overview

Start here for initialization, lifecycle, and tracing level controls.

## Initialization

### initialize(options)
Set up Laminar and OpenTelemetry (reads `LMNR_PROJECT_API_KEY` and `LMNR_BASE_URL` when options are omitted); call once per process.

**Parameters**
| Name | Type | Description |
| --- | --- | --- |
| `projectApiKey` | `string` | API key for the project (env fallback). |
| `baseUrl` | `string` | API base URL (env fallback). |
| `httpPort` / `grpcPort` | `number` | Override OTLP ports; defaults are 443/8443 (8000/8001 local). |
| `instrumentModules` | `Record<string, boolean \| object>` | Modules to auto-instrument. |
| `disableBatch` | `boolean` | Send spans immediately instead of batching. |
| `traceExportTimeoutMillis` | `number` | Export timeout (ms, default `30000`). |
| `traceExportBatchSize` / `maxExportBatchSize` | `number` | Control OTLP batch sizing. |
| `logLevel` | `'debug' \| 'info' \| 'warn' \| 'error'` | SDK logging level. |
| `forceHttp` | `boolean` | Force HTTP instead of HTTPS. |
| `sessionRecordingOptions` | `SessionRecordingOptions` | Browser session recording controls. |

**Returns:** `void`

**Example**
```ts
Laminar.initialize({
  projectApiKey: process.env.LMNR_PROJECT_API_KEY!,
  instrumentModules: { OpenAI },
});
```

### initialized()
Return `boolean` indicating whether Laminar has already been initialized.

## Lifecycle

### flush()
Force exporters to flush buffered spans.

**Returns:** `Promise<void>`

### shutdown()
Flush, reset configuration/context, and stop the Claude proxy.

**Returns:** `Promise<void>`

### getHttpUrl()
Return the configured HTTP URL (`string`).

### getProjectApiKey()
Return the configured project API key (`string | undefined`).

## Tracing Level

### withTracingLevel(level, fn)
Run `fn` with a tracing level override.

**Parameters**
| Name | Type | Description |
| --- | --- | --- |
| `level` | `TracingLevel` | `OFF`, `META_ONLY`, or `ALL` (default). |
| `fn` | `() => T` | Function to execute under the override. |

**Returns:** Result of `fn`

### TracingLevel enum
`ALL` (default), `META_ONLY` (metadata only), `OFF` (disable tracing).
