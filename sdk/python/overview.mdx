# SDK Overview

Initialization, lifecycle, and tracing level controls for Laminar's Python SDK.

## Initialization

### Laminar.initialize(...)
Set up Laminar and OpenTelemetry (reads `LMNR_PROJECT_API_KEY`, `LMNR_BASE_URL`, and OTEL env vars if options are omitted); call once per process.

**Parameters**
| Name | Type | Description |
| --- | --- | --- |
| `project_api_key` | `str` | API key for the project (env fallback). |
| `base_url` / `base_http_url` | `str` | API base URL and OTLP HTTP base. |
| `http_port` / `grpc_port` | `int` | OTLP port overrides. |
| `instruments` / `disabled_instruments` | `Iterable[Instruments]` | Select or block auto-instrumentations. |
| `disable_batch` | `bool` | Send spans immediately instead of batching. |
| `max_export_batch_size` | `int` | OTLP batch size cap. |
| `export_timeout_seconds` | `int` | OTLP export timeout. |
| `set_global_tracer_provider` | `bool` | Replace global tracer provider (default `True`). |
| `otel_logger_level` | `logging.Level` | Log level for OTEL internals. |
| `session_recording_options` | `SessionRecordingOptions` | Browser session recording controls. |
| `force_http` | `bool` | Force HTTP instead of HTTPS. |

**Returns:** `None`

## Lifecycle

### flush()
Flush spans (best-effort).

### force_flush()
Flush spans and reinitialize the pipeline (useful for serverless).

### shutdown()
Flush and shut down the pipeline.

### is_initialized()
Check whether Laminar has been initialized.

### get_base_http_url()
Return the configured OTLP HTTP URL.

### get_project_api_key()
Return the configured project API key (or `None`).

## Tracing Level

### TracingLevel enum
`ALL` (default), `META_ONLY` (metadata only), `OFF` (disable tracing in wrapper).

### set_tracing_level(level)
Context manager to override tracing level temporarily.

**Example**
```python
from lmnr import Laminar, TracingLevel

with Laminar.set_tracing_level(TracingLevel.META_ONLY):
    handle_sensitive_data()
```
