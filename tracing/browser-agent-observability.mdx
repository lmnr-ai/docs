## Overview

Laminar has an extensive observability suite for browser agents. If you are 
building browser agents, you can use Laminar to trace them and record the browser sessions.

That is, by default, you will get both the LLM calls traces and a session recording of
the browser sessions.

## How does it work?

Laminar traces the LLM calls using automatic instrumentations provided by OpenLLMetry.

In addition, Laminar instruments popular browser automation frameworks (integrations with [Puppeteer](/tracing/integrations/puppeteer), [Playwright](/tracing/integrations/playwright), [Stagehand](/tracing/integrations/stagehand), [BrowserUse](/tracing/integrations/browser-use), and [Skyvern](/tracing/integrations/skyvern)) and records the browser sessions.

## Example

Here is a Browser Use agent example (Python) and a Playwright example (JavaScript) that produce longer, interactive browser sessions (search, click, scroll).

<Tip>
If you don't have a project API key, you can get one by signing up on [Laminar](https://lmnr.ai) or spinning up a [self-hosted instance](https://github.com/lmnr-ai/lmnr) and getting a key from the project settings.
</Tip>

<Tabs>
<Tab title="Python (Browser Use)">

```python
# requires: pip install "lmnr" "browser-use" "langchain-openai" "playwright"
# and playwright install
import asyncio
from lmnr import Laminar
from browser_use import Agent
from langchain_openai import ChatOpenAI

Laminar.initialize(project_api_key="YOUR_LMNR_PROJECT_API_KEY")

async def main():
    agent = Agent(
        task="Open Wikipedia, search for 'laminar flow', click the article, scroll, and summarize the first paragraph.",
        llm=ChatOpenAI(model="gpt-4o-mini"),
    )
    result = await agent.run()
    print(result)

if __name__ == "__main__":
    asyncio.run(main())
```

</Tab>
<Tab title="JavaScript (Playwright)">

```javascript
// requires: npm install @lmnr-ai/lmnr playwright
//           npx playwright install
import { Laminar } from '@lmnr-ai/lmnr';
import { chromium } from 'playwright';

Laminar.initialize({
  projectApiKey: process.env.LMNR_PROJECT_API_KEY,
  instrumentModules: { playwright: { chromium } },
});

(async () => {
  const browser = await chromium.launch({ headless: true });
  const page = await browser.newPage();
  try {
    await page.goto('https://en.wikipedia.org');
    await page.getByLabel('Search Wikipedia').fill('Laminar flow');
    await page.keyboard.press('Enter');
    await page.waitForLoadState('domcontentloaded');
    await page.getByRole('heading', { level: 1 }).first().waitFor({ timeout: 8000 });
    for (let i = 0; i < 3; i += 1) {
      await page.mouse.wheel(0, 600);
      await page.waitForTimeout(700);
    }
    const tocLinks = page.locator('#toc a');
    const count = await tocLinks.count();
    for (let i = 0; i < Math.min(count, 2); i += 1) {
      await tocLinks.nth(i).click();
      await page.waitForTimeout(800);
    }
    await page.mouse.wheel(0, 1200);
    await page.waitForTimeout(800);
    await page.mouse.wheel(0, -600);
    await page.waitForTimeout(800);
    console.log('Page title:', await page.title());
  } finally {
    await browser.close();
    await Laminar.shutdown();
  }
})();
```
