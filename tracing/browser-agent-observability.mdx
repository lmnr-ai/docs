## Why this matters

Logs alone can’t tell you what the agent saw. Laminar records browser sessions alongside traces, so every agent step lines up with the exact frame in the recording.

## One-line setup

<CodeGroup>
```typescript TypeScript
import { Laminar } from '@lmnr-ai/lmnr';
import { chromium } from 'playwright';

Laminar.initialize({
  projectApiKey: process.env.LMNR_PROJECT_API_KEY,
  instrumentModules: { playwright: { chromium } },
});
```

```python Python
from lmnr import Laminar
Laminar.initialize(project_api_key="...")
# Browser Use, Stagehand, Playwright, Puppeteer, Skyvern are supported automatically
```
</CodeGroup>

That’s it. Browser sessions are recorded and synced to traces.

## Example: Browser Use agent

<CodeGroup>
```python Python
from langchain_anthropic import ChatAnthropic
from browser_use import Agent
from lmnr import Laminar

Laminar.initialize(project_api_key="...")

agent = Agent(
    task="Go to ycombinator.com, list 5 companies from the latest batch.",
    llm=ChatAnthropic(model="claude-3-5-sonnet-20241022")
)
result = agent.run()
print(result)
```
</CodeGroup>

Open your dashboard. Every agent step is a span; each span is synced to the recording.
