---
title: Metadata & Tags
---

Use metadata for key-value context (e.g., environment, region, feature flags) and post-hoc signals (scores, reviewer notes). Use tags for simple categorical labels. Both can be added while the trace runs and after it has ended.

- Metadata: [Metadata](/tracing/structure/metadata)
- Tags: [Tags](/tracing/structure/tags)
- SDK reference: [Client](/sdk/client), [Trace methods](/sdk/trace-methods), [Span methods](/sdk/span-methods)
