---

Sessions group related traces—useful for conversational AI, multi-step workflows, or tracking a user's journey through your product.

- Set a session ID at the trace level so all spans share it.
- In the Laminar UI, you can filter traces by session and review activity across a conversation or workflow.

Set session IDs early in the request/handler entry point so downstream spans inherit the context.

<Tabs items={['TypeScript', 'Python']}>
  <Tab title="TypeScript">
    ```typescript
    Laminar.setTraceSessionId('conversation-123');
    ```
  </Tab>
  <Tab title="Python">
    ```python
    Laminar.set_trace_session_id('conversation-123')
    ```
  </Tab>
</Tabs>
