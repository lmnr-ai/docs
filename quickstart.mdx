## 1) Install the SDK

<Tabs>
<Tab title="JavaScript/TypeScript">

```bash
npm add @lmnr-ai/lmnr
```
</Tab>
<Tab title="Python">

```bash
pip install --upgrade 'lmnr[all]'
```
</Tab>
</Tabs>

## 2) Get a project API key

<GetProjectApiKey />

Add it to your environment:

```env
LMNR_PROJECT_API_KEY=your_project_api_key_here
```

## 3) Initialize Laminar

Two lines enable tracing.

<Tabs>
<Tab title="JavaScript/TypeScript">

```javascript
import { Laminar } from '@lmnr-ai/lmnr';
import { OpenAI } from 'openai';

Laminar.initialize({
  projectApiKey: process.env.LMNR_PROJECT_API_KEY,
  instrumentModules: { openai: OpenAI },
});
```
</Tab>
<Tab title="Python">

```python
import os
from lmnr import Laminar

Laminar.initialize(project_api_key=os.environ["LMNR_PROJECT_API_KEY"])
```
</Tab>
</Tabs>

## 4) Make an LLM call

Your existing code is now traced automatically.

<Tabs>
<Tab title="JavaScript/TypeScript">

```javascript
import { OpenAI } from 'openai';

const client = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });
const response = await client.chat.completions.create({
  model: "gpt-4o-mini",
  messages: [{ role: "user", content: "What is the capital of France?" }],
});
```
</Tab>
<Tab title="Python">

```python
from openai import OpenAI

client = OpenAI()
response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "What is the capital of France?"}],
)
```
</Tab>
</Tabs>

Laminar captures prompts, responses, tokens, cost, and latency without extra code.
