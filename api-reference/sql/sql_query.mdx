---

Execute SQL against Laminar data (spans, traces, evals). Learn more in the [SQL Editor](/sql-editor/introduction) reference.

### Example request

```json
{
  "query": "SELECT * FROM spans WHERE start_time > now() - interval '1 hour' LIMIT 10"
}
```
