The Laminar Evaluations API allows you to programmatically create, manage, and update evaluations.

## Authentication

All API requests require authentication using your project API key in the `Authorization` header:

```bash
Authorization: Bearer your_project_api_key
```

## Base URL

https://api.lmnr.ai


## Overview

The evaluations API consists of three endpoints:

1. **Create Evaluation** - Initialize a new evaluation session
2. **Save Datapoints** - Add evaluation datapoints with input data and expected outputs
3. **Update Datapoint** - Update datapoint with execution results and scores

## API Endpoints

### 1. Create Evaluation

Initialize a new evaluation to start collecting datapoints and results.

<CardGroup cols={1}>
    <Card title="POST /v1/evals" icon="file" href="/api-reference/evals/init_eval">
        Create a new evaluation
    </Card>
</CardGroup>

**Example Request:**
```javascript
const client = new LaminarClient({ projectApiKey: 'YOUR_PROJECT_API_KEY' });

const evalId = await client.evals.createEvaluation(
  "Mixed Evaluator Q&A",
  "AI Testing Suite with Human Evaluation"
);
```

### 2. Save Evaluation Datapoints

Add datapoints to an existing evaluation with input data, expected outputs, and metadata.

<CardGroup cols={1}>
    <Card title="POST /v1/evals/{eval_id}/datapoints" icon="file" href="/api-reference/evals/save_eval_datapoints">
        Save evaluation datapoints to an evaluation
    </Card>
</CardGroup>

**Example Request:**
```javascript
const client = new LaminarClient({ projectApiKey: 'YOUR_PROJECT_API_KEY' });

const datapointId = await client.evals.createDatapoint({
  evalId,
  data: { question: "What is 2 + 2?" },
  target: { expectedAnswer: "4" },
  metadata: {
    difficulty: "easy",
    topic: "arithmetic",
    evaluator: 'human'
  },
  index: 0,
  traceId, // Optional: Link to tracing data
});
```

### 3. Update Evaluation Datapoint

Update an existing datapoint with execution results, scores, and evaluation outcomes.

<CardGroup cols={1}>
    <Card title="POST /v1/evals/{eval_id}/datapoints/{datapoint_id}" icon="file" href="/api-reference/evals/update_eval_datapoint">
        Update evaluation datapoint with scores and results
    </Card>
</CardGroup>

**Example Request:**
```javascript
const client = new LaminarClient({ projectApiKey: 'YOUR_PROJECT_API_KEY' });

await client.evals.updateDatapoint({
  evalId,
  datapointId,
  scores: {
    correctness: 0.95,
    relevance: 0.90,
    clarity: 0.85,
  },
  executorOutput: {
    response: "4",
    model: 'gpt-4o-mini',
    reasoning: "Human evaluator confirms this is correct",
    evaluator: 'human'
  },
});
```
