## Install the package

<Tabs>
<Tab title="JavaScript/Typescript">

Install the package from [npm](https://www.npmjs.com/package/@lmnr-ai/lmnr).

```sh
npm add @lmnr-ai/lmnr
```

<Expandable title="other installation options: yarn, pnpm">
yarn

```sh
yarn add @lmnr-ai/lmnr
```

pnpm

```sh
pnpm add @lmnr-ai/lmnr
```
</Expandable>
</Tab>
<Tab title="Python">

Install the package from [PyPI](https://pypi.org/project/lmnr/).

```sh
pip install --upgrade 'lmnr[all]'
```

This will install the package and enable all the available automatic instrumentations.

However, this installs a lot of dependencies, so you can specify the extras to enable
specific automatic instrumentations of client SDKs/libraries.

For example, to enable automatic instrumentations of Anthropic and OpenAI, run:

```sh
pip install --upgrade 'lmnr[anthropic,openai]'
```

<Warning>
If you do not specify any extras, no automatic instrumentation will be enabled.
</Warning>

<Expandable title="other installation options: poetry, uv">

poetry

```sh
poetry add 'lmnr[anthropic,openai]'
```

uv

```sh
uv add lmnr --extra anthropic --extra openai
```

uv pip

```sh
uv pip install 'lmnr[anthropic,openai]'
```

</Expandable>

- `bedrock`
- `cohere`
- `google-generativeai`
- `groq`
- `haystack`

- `milvus`
- `mistralai`
- `ollama`

- `qdrant`
- `replicate`
- `sagemaker`
- `together`

- `watsonx`
- `weaviate`


<Note>
  The extras configuration is only available since version `0.4.39`.
  Before that, default option would install all the available instruments.
</Note>
