<Tab title="JavaScript/Typescript">

Install the package from [npm](https://www.npmjs.com/package/@lmnr-ai/lmnr).

```sh
npm add @lmnr-ai/lmnr
```

<Expandable title="other installation options: yarn, pnpm">

```sh
yarn add @lmnr-ai/lmnr
```

pnpm

```sh
pnpm add @lmnr-ai/lmnr
```

<Tab title="Python">

Install the package from [PyPI](https://pypi.org/project/lmnr/). You can
specify the extras to enable automatic instrumentations of client SDKs/libraries.

For example, to enable automatic instrumentations of Anthropic and OpenAI, run:

```sh
pip install 'lmnr[anthropic,openai]'
```

<Warning>
If you do not specify the extras, no automatic instrumentation will be enabled.
</Warning>

- `watsonx`
- `weaviate`
</Expandable>

If you want to install all instruments at once with a special `all` extra:

```sh
pip install 'lmnr[all]'
```

Note that this will install all the available instruments, which might be a lot of dependencies.

<Note>
  The extras configuration is only available since version `0.4.39`. Before that default would install all the available instruments.
</Note>

<Expandable title="other installation options: poetry, uv">

poetry

```sh
poetry add 'lmnr[anthropic,openai]'
```

uv

```sh
uv add lmnr --extra anthropic --extra openai
```

uv pip

```sh
uv pip install 'lmnr[anthropic,openai]'
```
